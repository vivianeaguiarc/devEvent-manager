# API Documentation

## Overview

This project uses OpenAPI documentation with Swagger UI.

## Tools

- @fastify/swagger
- @fastify/swagger-ui

## Purpose

The API documentation exists to:

- document available endpoints
- standardize request and response contracts
- improve developer experience
- make the project easier to evaluate in a portfolio context

## Access

Swagger UI is available at:

`/docs`

## Notes

The OpenAPI document is generated from Fastify route schemas.
