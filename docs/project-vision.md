# Project Vision

## Overview

Dev Event Manager is a portfolio project designed to simulate a real-world backend system used to manage technology events.

The platform allows organizers to create events and developers to register, reserve spots and perform event check-in.

The main goal of the project is to demonstrate professional software engineering practices including:

- clean project structure
- code quality tooling
- scalable architecture
- strong TypeScript usage
- database modeling
- reproducible environments

## Target Audience

This project is designed for:

- developers interested in technology events
- event organizers
- recruiters evaluating backend engineering skills

## Main Goals

- demonstrate Node.js backend architecture
- implement a relational database with Prisma
- enforce code quality standards
- document architectural decisions
- simulate a real production-ready repository

## Non Goals

This project is not intended to:

- become a production platform
- handle real financial transactions
- support large scale deployments

The focus is educational and portfolio oriented.
